# Rank-Challenge 

## O que você precisa pra executar? 
- docker 
- docker-compose

## Como executar? 
- `git clone https://github.com/lucaspolizeli/rank-challenge`
- `cd rank-challenge`
- Abrir arquivo .env e colocar todas variáveis de ambiente solicitadas no arquivo. 
- `docker-compose up`

Após seguir os comandos acima, o serviço estará disponível em http://localhost:PORTA_QUE_VOCE_ESCOLHEU

## Ferramentas 
- Docker e docker-compose para containers
- Node.js para o backend
- Express para criar rotas. 
- MongoDB para banco de dados
